GoDaddy - Wild West PHP API

========================================== This PHP library connects with GoDaddy's Wild West Reseller API and performs all operations outlined in the pre-certification. This library requires zend framework to be in your include path. In fact, I have included all the scripts necessary to complete WildWest's pre certification. Use Cases: 1. Using WildWest_Reseller_Factory ------------------------------------------ $client = WildWest_Reseller_Factory::build(); 2. Checking Domain Name Availability ------------------------------------------ $client = WildWest_Reseller_Factory::build(); $results = $client->CheckAvailability(array( 'example.biz', 'example.us' )); // $results['example.biz'] = [0 = unavailable | 1 = available]; -- Need to Finish Readme Documentation
